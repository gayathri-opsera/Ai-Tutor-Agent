# Runbooks

Operational runbooks for the Ai Tutor Agent platform. Each runbook covers a specific failure scenario with diagnosis steps and resolution procedures.

## Index

| Runbook | Scenario |
|---------|----------|
| [chat-degraded](./chat-degraded.md) | Chat responses slow or returning demo answers |
| [rag-empty-results](./rag-empty-results.md) | RAG pipeline returning no chunks |
| [weaviate-down](./weaviate-down.md) | Weaviate vector DB unavailable |
| [db-connection-exhausted](./db-connection-exhausted.md) | PostgreSQL connection pool exhausted |
| [kafka-consumer-lag](./kafka-consumer-lag.md) | Kafka consumer group falling behind |
